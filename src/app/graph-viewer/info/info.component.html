Welcome to SPARQL Explorer!

This UI explores a knowledge graph dataset.

To get started either:

  • Select the menu icon and click Search to search for text in knowledge object properties and labels, or
  • Select the menu icon and click Schema to see a list of graph classes. You may be able to find knowledge linked to these classes, and use that as a starting point for graph exploration. Note that there may be classes in the schema with no linked information, so you should look for classes which you think exist in the dataset.

Once you have some graph nodes to work with, you can:

  • Click on a graph node to see properties in detail
  • Click on a node and press re-centre, this clears the entire graph leaving only the selected node. This can be useful if your graph display is cluttered and you want to continue exploring with less information on the screen.
  • Select a graph node, and click navigation buttons which appear with arrow icons e.g. type ⮕. This tells SPARQL Explorer to add nodes related to the selected object by following the relationships shown.

Note when exploring relationships, all relationships have a direction (indicated by the direction of arrow-heads on the lines. The direction of the arrow on the button indicates which direction to explore.

  • When the arrow icon is on the left of the button, SPARQL explorer will look for relationships which are 'incoming' i.e. the arrow-head points to the selected node.
  • When the arrow icon is on the right of the button, SPARQL explorer will look for relationships which are 'outgoing' i.e. the arrow-head points away from the selected node.

By way of an example, if you select a node and then press the button marked type ⮕, you will explore type relationships with the selected node at the start of the arrow. This is essentially asking "What type is this node?".

If you select a node and select the button marked ⮕ type, you will explore type relationships with the selected node at the end of the arrow. This is essentially asking "Show me all nodes which have this type?".
