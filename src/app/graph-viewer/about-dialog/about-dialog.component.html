SPARQL Sleuth is an open-source knowledge exploration tool for knowledge represented in RDF. It can search and explore a knowledge base by querying a knowledge graph which supports the SPARQL query language.

Also available by the same author is SPARQL Service, a simple RDF knowledge store, which allows RDF data to be queried using SPARQL.

Both tools are hosted on Github here:

https://github.com/cybermaggedon/sparql-sleuth

https://github.com/cybermaggedon/sparql-service
